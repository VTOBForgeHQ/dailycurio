# Purpose

This document defines the official DailyCurio color palette.

These colors represent the emotional and functional language of the product.

Colors should guide attention, establish hierarchy, and create a comfortable reading environment.

---

# Brand Direction

DailyCurio should feel:

- Curious
- Calm
- Intelligent
- Friendly
- Modern
- Minimal

The palette should support these emotions.

---

# Color Usage Rules

## Rule 1

Primary Blue should be used intentionally.

It should not appear everywhere.

---

## Rule 2

Text colors should prioritize readability.

Decoration must never reduce comprehension.

---

## Rule 3

Semantic colors should maintain consistent meaning.

Green always represents positive states.

Red always represents errors or danger.

---

## Rule 4

Additional colors require design approval.

New colors should only be introduced when a real product requirement exists.

---

# Future Expansion

Future versions may introduce:

- Category colors
- Achievement colors
- Premium features
- Educational levels

These additions should extend the system, not break consistency.
