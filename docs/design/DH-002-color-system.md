# Purpose

This document defines the official color language of DailyCurio.

Colors are communication tools.

Every color must have a clear purpose.

No color exists purely for decoration.

---

# Color Philosophy

DailyCurio is a calm reading experience.

The interface should quietly support learning.

Content should receive the greatest visual attention.

Colors should create clarity—not excitement.

# Color Rules

## Rule 1

Never use color alone to communicate meaning.

Always combine color with icons, text, or layout.

---

## Rule 2

Avoid unnecessary color variety.

Too many colors reduce clarity.

---

## Rule 3

Primary actions should always feel familiar.

Users should quickly recognize interactive elements.

---

## Rule 4

Backgrounds should remain visually quiet.

Content—not decoration—should attract attention.

---

## Rule 5

Contrast is mandatory.

Readability always wins over aesthetics.
